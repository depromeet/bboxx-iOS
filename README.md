## 개발 환경

- `Swift 5`
- Minimum Target Version: `iOS 14`
- Xcode `12.4` & `13.1`
- Dependency Manager: `CocoaPods` `Swift Package Manager`

## Rules

### Branch Rule 

develop: default   
       ⎿  feature-자기이름-기능명   
            ex: feature-keunna-settingArchitecture  
            ex: feature-eunwoo-measureDecibel  

### PR Rule
1. issue 열기   
2. PR 보낼 때 관련 이슈 #이슈번호 로 연결   
3. feature 개발 후 develop 브랜치를 향하여 Pull Request   
⎿ 24시간 안에 다른 iOS 개발자가 확인 후 코멘트 or approve    
4. PR 요청자가 스스로  merge
